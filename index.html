Before You Join

Here are our server rules. Please (please) don't do anything that would make us want to add another one.

Feel Free to Help

If you'd like to contribute, we need all the help we can get. Every block counts!

Treat Others with Respect

Please be kind; any kind of hateful or offensive conduct will not be tolerated.

Treat the Project with Respect

Do not intentionally disrupt or otherwise cause harm to the build.

You Can Have Fun
